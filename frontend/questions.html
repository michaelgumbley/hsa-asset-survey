Information Asset Survey - Questions

This survey is not optimised for small devices. Please use a desktop or laptop to complete it.

Key Information

What are we doing? The intent of this form is to gather initial information related to HealthServe Australia's digital assets, also known as information assets. We are creating an Information Asset Register as part of our Cyber Security assessment.

What are information assets? Any data, device or application used to access, modify or store HSA information. In our case this will likely include device and application assets that HSA does not own or license. (More on that as we go.)

If in doubt, add it. As you answer the following questions, opt to include things that you are unsure about. It is better for us to remove superfluous items later if they are not required, rather than miss important information.

A note on this process: This form is being distributed internally only - directly to staff, board members and volunteers who interact with HSA information. This survey is an activity that has been authorised by HSA's Board and CEO.

Survey Questions

QUESTION 1: Please list the DEVICES you use to access HSA systems, and to create, read, modify or delete HSA documents and other files.

Think about: laptops, PCs, tablets, phones, etc.

Example - Device: Windows Laptop, Owner: mine

Please list all devices you have used in the last 6 months.

QUESTION 2: Please list the APPLICATIONS you use to access, create, store and modify HSA information.

Think about: office software - MS Office, Google Docs, Google Drive, Dropbox, as well as software for accounting, website content management, video and image editing, video conferencing, etc.

No need to include applications that exist on a device regardless of HSA use, such as the operating system or virus protection software.

Example - Application: MS Office, Owner: mine, Usage: Working with HSA Word and Excel files

QUESTION 3: Please list all the DATA/FILES you access, create and modify as part of your role.

Think about: files you use and/or manage, as well as data related to the applications above promotional content.

There is no need to list every individual file - just list folder groups - e.g. board reports, financial reports, program proposals, website program content.

Example - Data/File Group: Financial Reports, Short Description: Created from Xero data to enable organisational governance
